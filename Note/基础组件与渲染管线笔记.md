# SoftRenderer 项目学习笔记

## 一、项目概述

这是一个用 C++ 实现的**软渲染器（Software Renderer）**，用于学习计算机图形学的基础知识。项目不依赖任何图形 API（如 OpenGL、DirectX），完全用 CPU 实现渲染管线。

### 技术栈
- **语言**：C++20
- **构建系统**：CMake 4.2+
- **并行计算**：OpenMP
- **输出格式**：TGA 图像

---

## 三、核心模块详解

### 3.1 TGA 图像类（tgaimage.h/cpp）

**作用**：TGAImage 是整个渲染器的"画布"，同时充当两个角色：
1. **帧缓冲（Framebuffer）**：渲染器的输出目标
2. **纹理（Texture）**：可加载贴图文件

#### TGA 文件格式要点
- 文件头固定 18 字节，使用 `#pragma pack(1)` 确保内存对齐
- 像素顺序是 **BGRA**（蓝-绿-红-透明），不是常见的 RGBA
- 支持未压缩和 RLE 压缩两种格式

#### 核心 API

```cpp
// 创建图像
TGAImage img(width, height, TGAImage::RGB);

// 读写像素
TGAColor color = img.get(x, y);      // 读取，越界返回全零
img.set(x, y, color);                // 写入，越界静默忽略

// 文件操作
img.read_tga_file("input.tga");
img.write_tga_file("output.tga");

// 翻转操作
img.flip_horizontally();
img.flip_vertically();
```

#### TGAColor 结构体
```cpp
struct TGAColor {
    std::uint8_t bgra[4] = {0,0,0,0};
    std::uint8_t bytespp = 4;
    std::uint8_t& operator[](const int i) { return bgra[i]; }
};
```

**颜色对照表**（BGRA 顺序）：
| 颜色 | bgra[0](B) | bgra[1](G) | bgra[2](R) | bgra[3](A) |
|------|-----------|-----------|-----------|-----------|
| 白色 | 255 | 255 | 255 | 255 |
| 红色 | 0 | 0 | 255 | 255 |
| 绿色 | 0 | 255 | 0 | 255 |
| 蓝色 | 255 | 0 | 0 | 255 |

#### RLE 压缩算法
- **RAW 块**：块头 < 128，后面跟 (块头+1) 个不重复像素
- **RLE 块**：块头 ≥ 128，后面 1 个像素重复 (块头-127) 次
- 最大块长度 128 像素

#### Vec4 模板类（齐次坐标）
```cpp
template <class T> struct Vec4 {
    T x, y, z, w;  // w 为齐次坐标分量

    Vec4(const Vec3<T>& v, T _w = 1);  // 从 Vec3 构造，默认 w=1（点）
    Vec3<T> xyz() const;                  // 截取前三分量

    Vec4<T> operator*(float scalar) const;

typedef Vec4<float> Vec4f;
```

**Vec4 与齐次坐标的关键方法**：
- `Vec4(Vec3f, 1)` — 表示点（w=1），平移有效
- `Vec4(Vec3f, 0)` — 表示方向向量（w=0），平移无效
- `toVec3()` — 执行透视除法，将齐次坐标转回三维坐标；w=0 时直接截取 xyz

#### Matrix 模板类
```cpp
template <size_t ROWS, size_t COLS, typename T>
struct Matrix {
    std::array<std::array<T, COLS>, ROWS> m{};  // 行主序存储

    Matrix() = default;
    std::array<T, COLS>& operator[](size_t i);       // 按行访问
    const std::array<T, COLS>& operator[](size_t i) const;

    template <size_t C2>
    Matrix<ROWS, C2, T> operator*(const Matrix<COLS, C2, T>& other) const;  // 矩阵乘法

    Matrix<COLS, ROWS, T> transpose() const;    // 转置
    Matrix<ROWS-1, COLS-1, T> minor(size_t r, size_t c) const;  // 余子式
    T det() const;                                // 行列式（仅方阵）
};
typedef Matrix<2, 2, float> Mat2;
typedef Matrix<3, 3, float> Mat3;
typedef Matrix<4, 4, float> Mat4;
```

**矩阵与向量乘法**：
```cpp
Vec2f r = mat2 * v2;                   // 2×2 × Vec2
Vec3f r = mat3 * v3;                   // 3×3 × Vec3
Vec4f r = mat4 * v4;                   // 4×4 × Vec4
Vec3f r = mat4 * v3;                   // 4×4 × Vec3 → 自动视为 (x,y,z,1)，执行透视除法
```

**工具函数**：
| 函数 | 说明 |
|------|------|
| `identity_matrix<N, T>()` | 生成 N×N 单位矩阵 |
| `inverse(mat)` | 矩阵求逆（高斯-约当消元法），奇异时返回零矩阵 |
| `invert_transpose(mat)` | 逆转置 `(A⁻¹)ᵀ`，用于法线变换和重心坐标计算 |
| `rotation_x/y/z(angle)` | 绕 X/Y/Z 轴旋转的 4×4 矩阵（弧度） |
| `rotation_axis(axis, angle)` | 绕任意轴旋转（Rodrigues 公式） |
| `rotation_2d(angle)` | 2D 旋转的 3×3 矩阵 |

**数学公式**：
- 叉积：`(a × b) = (a.y*b.z - a.z*b.y, a.z*b.x - a.x*b.z, a.x*b.y - a.y*b.x)`
- 点积：`a · b = a.x*b.x + a.y*b.y + a.z*b.z`
- 向量长度：`|v| = √(x² + y² + z²)`

### 3.3 OBJ 模型加载器（Model.h/cpp）

负责从 `.obj` 文件中解析 3D 模型数据。

#### OBJ 文件格式
```
# 注释
v x y z              # 顶点坐标
f v1 v2 v3           # 面（三角形）
f v1/vt1/vn1 ...     # 面（带纹理坐标和法线）
f v1//vn1 ...        # 面（带法线，无纹理）
```

#### Model 类接口
```cpp
class Model {
private:
    std::vector<Vec3f> verts_;
    std::vector<std::vector<int>> faces_;

public:
    Model(const char *filename);
    Model(const std::string &filename);
    int nverts() const;
    int nfaces() const;
    Vec3f vert(int i) const;
    std::vector<int> face(int idx) const;
};
```

#### 解析逻辑
1. 逐行读取文件
2. 以 `"v "` 开头 → 解析顶点坐标
3. 以 `"f "` 开头 → 解析面索引（OBJ 索引从 1 开始，需转换为 0-based）
4. 自动跳过纹理坐标和法线索引

### 3.4 渲染器核心（renderer.h/cpp）

渲染管线中的 MVP 矩阵构建、三角形光栅化和 Z 缓冲都在此模块实现。

#### 3.4.1 全局状态

```cpp
Mat4 modelview;             // 模型-视图矩阵（世界→相机空间）
Mat4 perspective;            // 透视投影矩阵
Mat4 viewport;               // 视口变换矩阵（NDC→屏幕）
std::vector<float> zbufferf; // 浮点深度缓冲
```

#### 3.4.2 ModelView 矩阵 — `lookat` 变换

```cpp
void ModelView(const Vec3f eye, const Vec3f center, const Vec3f up) {
    Vec3f n = (eye - center).normalize();   // 相机前向（视线反方向）
    Vec3f l = (up ^ n).normalize();         // 相机右向
    Vec3f m = (n ^ l).normalize();          // 相机上向

    Mat4 rot = {
        {l.x, l.y, l.z, 0},     // l 行：右向分量
        {m.x, m.y, m.z, 0},     // m 行：上向分量
        {n.x, n.y, n.z, 0},     // n 行：前向分量
        {0,   0,   0,   1}
    };
    Mat4 trans = {
        {1, 0, 0, -center.x},
        {0, 1, 0, -center.y},
        {0, 0, 1, -center.z},
        {0, 0, 0, 1}
    };
    modelview = rot * trans;    // 先平移到原点，再旋转到相机基
}
```

**参数**：
- `eye`：相机位置（世界坐标）
- `center`：相机看向的目标点
- `up`：大致的上方向

#### 3.4.3 Perspective 矩阵

```cpp
void Perspective(float f) {
    perspective[0] = {1, 0, 0, 0};
    perspective[1] = {0, 1, 0, 0};
    perspective[2] = {0, 0, 1, 0};
    perspective[3] = {0, 0, -1/f, 1};
}
```

对齐次坐标点 `(vx, vy, vz, 1)` 做乘法后得到 `(vx, vy, vz, 1 - vz/f)`，透视除法后：
```
(vx, vy, vz) / (1 - vz/f)
```
`f` 为焦距，通常取 `(eye - center).norm()`。

#### 3.4.4 Viewport 矩阵

```cpp
void Viewport(int x, int y, int w, int h) {
    viewport[0] = {w/2.f, 0, 0, x+w/2.f};
    viewport[1] = {0, h/2.f, 0, y+h/2.f};
    viewport[2] = {0, 0, 1, 0};
    viewport[3] = {0, 0, 0, 1};
}
```

将 NDC 的 `[-1, 1]` 映射到屏幕上 `(x, y, w, h)` 指定的区域。

#### 3.4.5 Shader 接口（IShader）

```cpp
class IShader {
public:
    virtual std::pair<bool,TGAColor> fragment(const Vec3f bar) const = 0;
};
```

- `Vertex(face, vert)`：逐顶点着色器，返回经过 MVP 变换后的齐次坐标 `Vec4f`
- `fragment(bar)`：逐片元着色器，输入重心坐标 `bar`，返回 `{是否丢弃, 颜色}`

**当前示例 — RandomShader**：
```cpp
class RandomShader : public IShader {
public:
    const Model *model;
    TGAColor color;
    Vec3f tri[3];  // 存储模型空间→相机空间后的三个顶点（用于光照）

    virtual Vec4f Vertex(const int face, const int vert) {
        Vec4f v = {model->vert(model->face(face)[vert]), 1};
        tri[vert] = (modelview * v).toVec3();       // 保存相机空间顶点
        return perspective * modelview * v;           // 返回 Clip Space 坐标
    }

    virtual std::pair<bool,TGAColor> fragment(const Vec3f bar) const {
        return {false, color};    // 不丢弃，使用固定颜色
    }
};
```

#### 3.4.6 三角形光栅化 — Rasterize

```cpp
void Rasterize(TGAImage &img, const Vec4f face[3], const IShader &shader) {
    // 1. 透视除法 → NDC
    Vec4f ndc[3] = {
        Vec4f(face[0].x/face[0].w, face[0].y/face[0].w, face[0].z/face[0].w, 1),
        Vec4f(face[1].x/face[1].w, face[1].y/face[1].w, face[1].z/face[1].w, 1),
        Vec4f(face[2].x/face[2].w, face[2].y/face[2].w, face[2].z/face[2].w, 1),
    };
    // 2. 视口变换 → 屏幕坐标
    for (int i=0; i<3; i++) ndc[i] = viewport * ndc[i];
    Vec2f screen[3] = {Vec2f(ndc[0].x, ndc[0].y), ...};

    // 3. 构建 3×3 矩阵求重心坐标（矩阵法）
    Mat3 area;
    area[0] = {screen[0].x, screen[0].y, 1};
    area[1] = {screen[1].x, screen[1].y, 1};
    area[2] = {screen[2].x, screen[2].y, 1};
    float triangleArea = area.det();
    if (triangleArea < 1) return;

    // 4. 遍历包围盒内每个像素
    #pragma omp parallel for
    for (int x = minx; x <= maxx; x++) {
        for (int y = miny; y <= maxy; y++) {
            Vec3f heavyPoint = invert_transpose(area) * Vec3f(x, y, 1);
            if (heavyPoint.x < 0 || heavyPoint.y < 0 || heavyPoint.z < 0) continue;
            float z = heavyPoint * Vec3f{ndc[0].z, ndc[1].z, ndc[2].z};
            if (z <= zbufferf[x+y*img.width()]) continue;
            auto [discard, color] = shader.fragment(heavyPoint);
            if (discard) continue;
            zbufferf[x+y*img.width()] = z;
            img.set(x, y, color);
        }
    }
}
```

**重心坐标的矩阵法**：
- 将 3 个屏幕空间顶点排成 3×3 矩阵 `area`
- 用 `invert_transpose(area) * Vec3f(x, y, 1)` 直接算出重心坐标 `(α, β, γ)`
- 本质是解线性方程组，比面积法更高效且数学等价

#### 3.4.7 浮点深度缓冲

```cpp
std::vector<float> zbufferf;  // 初始化为 -1e10f
```

相比旧版用 `TGAImage` 做灰度 Z-buffer（精度仅 8-bit），新版使用 `float` 类型深度缓冲，精度更高。

**深度可视化**：通过 `CreateZBufferImage()` 将浮点深度映射到 0-255 灰度输出。

### 3.5 主程序（main.cpp）

#### 渲染流程

```
1. 设置画布尺寸和相机参数
       ↓
2. 初始化 Z-buffer
       ↓
3. 加载 OBJ 模型
       ↓
4. 构建 MVP 矩阵（ModelView / Perspective / Viewport）
       ↓
5. 对每个面：Shader.Vertex() → Rasterize()
       ↓
6. 输出 TGA 图像 + Z-buffer 图像
```

#### 主函数
```cpp
int main() {
    int width = 800, height = 800;
    Vec3f eye(-1,0,2), center(0,0,0), up(0,1,0);

    zbufferf.assign(width * height, -1e10f);
    TGAImage img(width, height, TGAImage::RGB);

    Model model(path);

    // MVP 矩阵
    ModelView(eye, center, up);
    Perspective((eye - center).norm());
    Viewport(width/16, height/16, width*7/8, height*7/8);

    RandomShader shader(&model);
    for (int i = 0; i < model.nfaces(); i++) {
        shader.color = RandomColor();
        Vec4f face[3] = {shader.Vertex(i,0), shader.Vertex(i,1), shader.Vertex(i,2)};
        Rasterize(img, face, shader);
    }

    img.write_tga_file("output.tga");
    CreateZBufferImage(zbufferf, width, height).write_tga_file("zbuffer.tga");
}
```

```

│                        渲染管线流程                                │

│                                                                  │
│  ┌──────────┐  ┌──────────┐  ┌───────────┐  ┌──────────┐       │
│  │ 加载模型  │→│ MVP 变换  │→│ 光栅化    │→│ 输出     │       │
│  │ (OBJ)    │  │ (矩阵)   │  │ (三角形+  │  │ (TGA)    │       │
│  │          │  │          │  │  Z-buffer) │  │          │       │
│  └──────────┘  └──────────┘  └───────────┘  └──────────┘       │
│                                                                  │
│  MVP 变换细部：                                                   │
│  ┌──────────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐    │
│  │ Object Space │→│ Eye Space │→│Clip Space │→│NDC Space │    │
│  │              │  │          │  │          │  │          │    │
│  │              │  │          │  │透视除法    │  │          │    │
│  │  ModelView   │  │PERSPECTIVE│  │ ÷ w      │  │VIEWPORT  │    │
│  │  × (v, 1)   │  │× clip    │  │          │  │× ndc     │    │
│  └──────────────┘  └──────────┘  └──────────┘  └──────────┘    │
│                                                                  │
│ clip = Perspective × ModelView × Vec4(v, 1)                     │
│ ndc  = clip / clip.w                                             │
│ screen = Viewport × ndc                                          │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

## 六、与旧版代码的对比

### 旧版（已注释）
```cpp
// 分步函数调用，手算投影
Vec3f v0 = persp(rot * model.vert(face[0]));
auto [x0, y0, z0] = project(v0);
Triangle(img, zbufferf, v0, v1, v2, x0, y0, x1, y1, x2, y2, color);
```

### 新版
```cpp
// 统一矩阵变换 + Shader 接口
Vec4f face[3] = {shader.Vertex(i,0), shader.Vertex(i,1), shader.Vertex(i,2)};
Rasterize(img, face, shader);
```

| 项目 | 旧版 | 新版 |
|------|------|------|
| 变换方式 | `persp()` + `project()` 函数 | `Perspective × ModelView` 矩阵乘法 |
| 投影 | 近似透视 | 标准透视除法（齐次坐标÷w） |
| 视口映射 | `project()` 手算 | `Viewport` 矩阵 |
| 相机 | `rotation_y()` 硬编码旋转 | `lookat()` 自由摆放相机 |
| 重心坐标 | 面积比计算 | `invert_transpose(area) * Vec3f(x,y,1)` 矩阵法 |
| Z-buffer | `TGAImage` 灰度图（8-bit） | `std::vector<float>` 浮点缓冲 |
| 着色 | 固定随机颜色 | `IShader` 接口，可扩展 Vertex/fragment |
| 深度值范围 | `[0, 255]` 灰度 | 任意浮点范围 |
